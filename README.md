**AI-Driven Parameterized Phishing Email Generator**

An open-source tool for cybersecurity awareness training. Generates realistic, context-aware phishing simulations using AI and rule-based indicators.

## Features
- **AI-Powered:** Uses Google Gemini (Free) or a built-in Fallback Engine.
- **Parameterized:** Customize emotional triggers (Feat, Greed, Urgency) and contexts (IT, HR, Finance).
- **Indicators:** Embeds measurable red flags like typosquatting, double extensions, and urgent language.
- **Privacy-Focused:** Runs 100% locally. No data leaves your machine.

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py` (or click `Start_PhishGuard.bat`)
3. Open `http://localhost:5000`
